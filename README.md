# 英熟語マスター 🗣️ (Idiom Master)

英会話に役立つ**英熟語（イディオム）**を、イメージで覚えて、文章の中で使えるようになるための学習アプリです。ビルド不要で、ブラウザで開くだけで動きます。

## 機能

**① レベルを選ぶ → ② モードを選ぶ → 学習 / ゲーム** という流れです。

レベルは2ジャンルに分かれています：📘 **英熟語**（中学／高校1・2年／高校3年・大学1年）と、💼 **TOEIC単語**（Lv1〜Lv5）。どちらも同じ全モード（カード／穴埋め／バトル）で遊べます。TOEIC単語は頻出のオリジナル単語集（独自の例文つき）で、各レベル200語を目標に拡充していきます。

### 🎚️ 難易度レベル
- **中学レベル** / **高校1・2年** / **高校3年・大学1年** の3段階
- レベルごとに熟語を **10問ずつ** のセットに自動分割。1セット終えると「次の10問へ」で進める

### 📇 フラッシュカード（十問ずつ）
- 選んだレベルの英熟語を **10問ずつ** のセットで学習
- 各カードに意味を表した**イラスト（SVG画像）**、無ければ**絵文字**を表示
- カードをクリックすると裏返り、意味・例文・日本語訳が見られる

### ✍️ 穴埋め入力
- 学んだ熟語を例文の空欄に**自分でタイプして**当てはめる入力式の問題
- 大文字小文字や余分な空白は無視して正誤判定
- **似た意味の別解も正解**にする（例: `look for` に `search for`、`give up` に `give in`）。バトルでも同様。別解は [idioms.js](idioms.js) の `SYNONYM_GROUPS` で管理
- スコアが記録され、最後に結果が出る

### ⚔️ 英熟語バトル（ローグライク・全100階）
- 空欄の**英熟語をタイプして正解すると敵にダメージ**、**不正解だと自分がHPを失う**
- **全100階**。10階ごとにボス、100階の最終ボス（魔王）を倒すとゲームクリア👑
- 通常の敵を倒すたびに**3つのごほうび（パーク）から1つ選んで強化**
  （攻撃アップ／会心の一撃／防御／最大HP+25／回復／吸収）
- **5階ごとに宝箱**が出現。**名前付きの装備を3択から1つ選んで装備**できる
  - スロット: ⚔️剣 / 🪖兜 / 👕鎧 / 👖ズボン / 👟靴
  - **レア度5段階**（コモン→アンコモン→レア→エピック→レジェンダリー）。深い階ほど良い装備が出やすい
  - 装備は**効果（fx）**を持ち、効果エンジンが解釈する。例: 攻撃/防御/最大HP、撃破ごとに強化、HP割合で攻撃up、ステータス変換、ボス特効、即死、復活、反射、回避、2回攻撃 など
  - 装備データは [equipment.js](equipment.js)。「レベルアップ」＝階を進む、「ターン」＝1問 と読み替えて実装
- 自分のHPが0でゲームオーバー。到達階・撃破数・**間違えた回数**が出て、何度でも挑戦できる
- 選んだレベルの熟語が出題されるので、遊びながら復習になる
- **バトル中だけBGM**が流れる（Web Audio APIで生成するチップチューン風・外部ファイル不要）。ヘッダーの🔊ボタンで**オン/オフ**でき、設定は保存される（[music.js](music.js)）

## 使い方

JS のビルドツールは不要です。次のどちらかで開けます。

```bash
# 方法1: ファイルをそのままブラウザで開く
open index.html

# 方法2: 簡易サーバーで配信する（推奨）
python3 -m http.server 8000
# → ブラウザで http://localhost:8000 を開く
```

## テスト

外部ライブラリ不要。Node 標準のテストランナーで動きます。

```bash
npm test   # 内部的には node --test
```

テスト内容（[test/app.test.js](test/app.test.js)）:
- 熟語データの件数・必須フィールド・重複チェック
- 例文に熟語が含まれているか（穴埋め問題が生成できるか）
- ロジック（`chunk` / `makeBlank` / `isCorrect` / `normalize`）の検証

## 英熟語を追加するには

[idioms.js](idioms.js) の `IDIOMS` 配列に追記するだけです。`level` でレベルを指定し、`example` には `phrase` を**そのままの形**で含めてください（穴埋め問題はそこから自動生成されます）。各レベル10件を超えると、自動的に10問ずつのセットに分割されます。

```js
{
  level: "junior", // "junior"（中学）/ "high12"（高校1・2年）/ "high3"（高校3年・大学1年）
  phrase: "break a leg",
  image: "images/break-a-leg.svg", // 任意。無い場合は emoji で表示される
  emoji: "🎭",
  meaning: "頑張って、成功を祈る",
  example: "Break a leg in your performance tonight!",
  exampleJa: "今夜の公演、頑張ってね！",
}
```

`images/` に対応する SVG を置くとイラストで表示され、無ければ `emoji` が代わりに表示されます。例文には必ず `phrase` をそのまま含めてください（テストでも検証されます）。
